## Config

```javascript
simulation.setConfig({
  brightness: 0.3,
  backgroundColor: "#000000,
  colorPalette: ["#FFFFFF"],
  inverted: true,
});
```

Try config in playground <a href="/play?colorPalette=%23FFFFFF&inverted=true&transparent=false&brightness=0.3" target="_self">here</a>
