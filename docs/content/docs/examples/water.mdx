## Config

```javascript

  densityDissipation: 0.5,
  velocityDissipation: 0,
  pressure: 0.7,
  curl: 5,
  colorPalette: ["#00FFFF","#00DBFF","#10A5F5","#0C71E0","#0859C6"],
  bloom: false,
});
```
